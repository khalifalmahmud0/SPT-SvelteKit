## Installing the project

```bash
# Clone git repository
git@github.com:khalifalmahmud0/SPT-SvelteKit.git
```

## Run the project

```bash
cd SPT-SvelteKit
yarn  dev
# or start the server and open the app in a new browser tab
yarn dev -- --open
```
